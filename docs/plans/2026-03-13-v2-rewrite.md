# tall-datatables v2 Rewrite Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rewrite the tall-datatables Livewire package to move state management, formatting, and filter logic from ~1,435 lines of Alpine.js to PHP, while keeping the public PHP API stable.

**Architecture:** Server-side rendering via Livewire v4. DataTable is the main component rendering table/rows/cells from `$this->data`. DataTableFilters and DataTableOptions are lazy child components dispatching events upward. Formatting handled by standalone PHP Formatter classes via FormatterRegistry. JS reduced to ~100-200 lines for DOM-only interactions (drag/drop, sticky scroll, keyboard shortcuts).

**Tech Stack:** PHP 8.4, Laravel 12, Livewire 4, TallStackUI 3, Alpine.js 3 (minimal), Pest 4, Orchestra Testbench

**Spec:** `docs/specs/2026-03-13-v2-rewrite-design.md`

**Testing:** Run all tests with `vendor/bin/testbench package:test` from the package root. Filter: `vendor/bin/testbench package:test --filter=TestName`.

```php
<?php

namespace TeamNiftyGmbH\DataTable\Formatters;

use TeamNiftyGmbH\DataTable\Formatters\Contracts\Formatter;

class MoneyFormatter implements Formatter
{
    public function __construct(
        public bool $colored = false,
    ) {}

    public function format(mixed $value, array $context = []): string
    {
        if (is_null($value)) {
            return '';
        }

        $value = (float) $value;
        $currencyCode = e($context['currency.iso'] ?? $context['currency_code'] ?? 'EUR');
        $locale = app()->getLocale();

        $formatted = $this->formatNumber($value, $currencyCode, $locale);

        if ($this->colored && $value != 0) {
            $colorClass = $value > 0 ? 'text-green-600' : 'text-red-600';

            return '<span class="' . $colorClass . '">' . $formatted . '</span>';
        }

        return $formatted;
    }

    protected function formatNumber(float $value, string $currencyCode, string $locale): string
    {
        if (class_exists(\NumberFormatter::class)) {
            $fmt = new \NumberFormatter($locale, \NumberFormatter::CURRENCY);

            return $fmt->formatCurrency($value, $currencyCode);
        }

        return number_format($value, 2, ',', '.') . ' ' . $currencyCode;
    }
}
```

```php
use TeamNiftyGmbH\DataTable\Formatters\Contracts\Formatter;

class FormatterRegistry
{
    /** @var array<string, Formatter> */
    protected array $formatters = [];

    protected array $autoDetected = [
        'boolean' => BooleanFormatter::class,
        'bool' => BooleanFormatter::class,
        'date' => DateFormatter::class,
        'datetime' => DateFormatter::class,
        'timestamp' => DateFormatter::class,
        'immutable_date' => DateFormatter::class,
        'immutable_datetime' => DateFormatter::class,
    ];

    public function register(string $castClass, Formatter $formatter): static
    {
        $this->formatters[$castClass] = $formatter;

        return $this;
    }

    public function resolve(string $castClass): Formatter
    {
        if (isset($this->formatters[$castClass])) {
            return $this->formatters[$castClass];
        }

        $normalized = strtolower(class_basename($castClass));

        if (isset($this->autoDetected[$normalized])) {
            return new ($this->autoDetected[$normalized]);
        }

        // Check if the cast is a known native type
        if (isset($this->autoDetected[$castClass])) {
            return new ($this->autoDetected[$castClass]);
        }

        return new StringFormatter();
    }

    public function resolveForColumn(string $column, array $casts): Formatter
    {
        $cast = $casts[$column] ?? null;

        if (is_null($cast)) {
            return new StringFormatter();
        }

        // Handle cast class strings like "date:Y-m-d" → extract base class
        $castClass = str_contains($cast, ':') ? explode(':', $cast, 2)[0] : $cast;

        return $this->resolve($castClass);
    }
}
```

```php
<?php

namespace TeamNiftyGmbH\DataTable\Support;

use Illuminate\Database\Eloquent\Model;
use TeamNiftyGmbH\DataTable\Formatters\FormatterRegistry;

class RowTransformer
{
    public function __construct(
        protected FormatterRegistry $registry,
    ) {}

    public function transform(Model $model, array $enabledCols): array
    {
        $context = $model->attributesToArray();
        $row = [];

        foreach ($enabledCols as $col) {
            $raw = data_get($model, $col);
            $colCasts = $this->resolveCasts($model, $col);
            $formatter = $this->registry->resolveForColumn(
                $this->baseColumn($col),
                $colCasts,
            );

            $display = $formatter->format($raw, $context);
            $rawString = is_null($raw) ? '' : (string) $raw;

            // Omit display if it matches raw to save payload
            if ($display === e($rawString)) {
                $row[$col] = ['raw' => $raw];
            } else {
                $row[$col] = ['raw' => $raw, 'display' => $display];
            }
        }

        return $row;
    }

    /**
     * Resolve the correct casts array for a column.
     * For relation columns (dot notation), look up casts on the related model.
     * For root columns, use the root model's casts.
     */
    protected function resolveCasts(Model $model, string $col): array
    {
        if (! str_contains($col, '.')) {
            return $model->getCasts();
        }

        // For "user.name" → resolve User model's casts
        $parts = explode('.', $col);
        array_pop($parts); // Remove the actual column
        $relation = implode('.', $parts);

        try {
            $related = $model;
            foreach (explode('.', $relation) as $segment) {
                $related = $related->{$segment}()->getRelated();
            }

            return $related->getCasts();
        } catch (\Throwable) {
            return [];
        }
    }

    protected function baseColumn(string $col): string
    {
        return str_contains($col, '.') ? last(explode('.', $col)) : $col;
    }
}
```

```php
<?php

namespace TeamNiftyGmbH\DataTable\Filters;

use Illuminate\Database\Eloquent\Builder;

class FilterApplier
{
    public function apply(Builder $query, array $filters): Builder
    {
        foreach ($filters as $filter) {
            $query = $this->addFilter($query, $filter);
        }

        return $query;
    }

    protected function addFilter(Builder $query, array $filter): Builder
    {
        $column = $filter['column'];
        $operator = $filter['operator'];
        $value = $filter['value'];

        // Handle relation columns (dot notation → whereHas)
        if (str_contains($column, '.')) {
            return $this->addRelationFilter($query, $column, $operator, $value);
        }

        return match ($operator) {
            'like' => $query->where($column, 'like', $value),
            'between' => $query->whereBetween($column, $value),
            'is null' => $query->whereNull($column),
            'is not null' => $query->whereNotNull($column),
            default => $query->where($column, $operator, $value),
        };
    }

    protected function addRelationFilter(Builder $query, string $column, string $operator, mixed $value): Builder
    {
        $parts = explode('.', $column);
        $field = array_pop($parts);
        $relation = implode('.', $parts);

        return $query->whereHas($relation, function (Builder $q) use ($field, $operator, $value) {
            $this->addFilter($q, [
                'column' => $field,
                'operator' => $operator,
                'value' => $value,
            ]);
        });
    }
}
```

```php
namespace TeamNiftyGmbH\DataTable\Traits\DataTables;

use Illuminate\Database\Eloquent\Builder;
use TeamNiftyGmbH\DataTable\Filters\FilterApplier;
use TeamNiftyGmbH\DataTable\Filters\FilterParser;

trait BuildsQueries
{
    protected function buildSearch(Builder $query): Builder
    {
        // Apply text filters from per-column inputs
        $parser = new FilterParser();
        $applier = new FilterApplier();

        $filters = [];
        foreach ($this->textFilters as $column => $text) {
            if ($filter = $parser->parse($text, $column)) {
                $filters[] = $filter;
            }
        }

        // Apply structured filters from sidebar
        $filters = array_merge($filters, $this->userFilters);

        return $applier->apply($query, $filters);
    }

    protected function getResultFromQuery(Builder $query): mixed
    {
        // Sort
        if ($this->orderBy) {
            $query->orderBy($this->orderBy, $this->orderAsc ? 'asc' : 'desc');
        }

        // Scout search integration
        if ($this->search && $this->isSearchable()) {
            return $this->applyScoutSearch($query);
        }

        return $query->paginate($this->perPage, ['*'], 'page', $this->page);
    }
}
```

- [ ] **Step 1: Rewrite HasEloquentListeners**

Move Echo channel subscription management from JS to PHP. The trait now uses Livewire's `getListeners()` to register Echo event handlers.

**Channel subscription timing:** `getListeners()` is called during component hydration before data loads, so `$this->broadcastChannels` may be empty on initial mount. Solution: channels are populated in `mount()` (based on the model class), and `getListeners()` can return them. Alternatively, use Livewire v4's dynamic listener registration (`$this->listenTo(...)`) if available.

```php
// NEW approach: Livewire-native Echo listeners
public function getListeners(): array
{
    $listeners = [];

    // broadcastChannels populated in mount() from model's broadcast config
    if ($this->broadcastChannels) {
        foreach ($this->broadcastChannels as $channel) {
            $listeners["echo-private:{$channel},.created"] = 'refreshData';
            $listeners["echo-private:{$channel},.updated"] = 'refreshRow';
            $listeners["echo-private:{$channel},.deleted"] = 'refreshData';
        }
    }

    return $listeners;
}

public function refreshRow(array $event): void
{
    $modelId = $event['model']['id'] ?? null;
    if (! $modelId) return;

    $model = $this->getBuilder()->find($modelId);
    if (! $model) return;

    $transformer = new RowTransformer($this->getFormatterRegistry());
    $row = $transformer->transform($model, $this->enabledCols);

    // Replace the row in $this->data
    $this->replaceRowInData($modelId, $row);
}

public function refreshData(): void
{
    $this->loadData();
}
```

## Implementation Order & Dependencies

```
Chunk 1 (Formatters)          ← No dependencies, fully independent
    │
    ▼
Chunk 2 (Support Classes)     ← Depends on Chunk 1 (FormatterRegistry)
    │
    ▼
Chunk 3 (Trait Refactoring)   ← Depends on Chunks 1 + 2
    │
    ▼
Chunk 4 (Child Components)    ← Depends on Chunk 3 (DataTable events)
    │
    ▼
Chunk 5 (Blade Views)         ← Depends on Chunks 3 + 4
    │
    ▼
Chunk 6 (JS & Integration)    ← Depends on all previous chunks
```

**Parallelizable:**
- Chunk 1 tasks (all formatters) can be developed in parallel
- Chunk 2 tasks (FilterParser, FilterApplier, ColumnResolver, RowTransformer) can be partially parallelized
- Chunk 5 tasks (individual views) can be partially parallelized

**Sequential:**
- Chunk 3 must complete before Chunk 4
- Chunk 4 must complete before Chunk 5 (child component views need the components)
- Chunk 6 is the integration phase and must be last
